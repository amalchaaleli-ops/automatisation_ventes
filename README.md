# 📊 Automatisation des Ventes - PFA Logiciels

## 🎯 Contexte du Projet
[cite_start]Ce script Python a été développé pour automatiser l'analyse des ventes d'une entreprise de e-commerce, résolvant ainsi la problématique du volume de données qui devient trop important pour un tableur classique[cite: 7, 8].

## ⚙️ Fonctionnalités
- [cite_start]**Génération** : Création d'un dataset de ventes au format CSV[cite: 12].
- [cite_start]**Analyse Pandas** : Calcul automatisé du Chiffre d'Affaires Brut, du CA Net (après remises), et de la TVA (20%)[cite: 18, 19, 20].
- [cite_start]**Business Intelligence** : Identification automatique de l'ID du produit générant le plus gros bénéfice[cite: 22].
- [cite_start]**Data Visualization** : Génération de graphiques avec Matplotlib pour visualiser le CA par produit[cite: 26].

## 🛠️ Stack Technique
- **Langage** : Python 
- **Bibliothèques** : Pandas, Matplotlib
- **Outils** : VS Code, Git, GitHub
